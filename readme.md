## Website ini dibuat menggunakan hugo, render.com & forestry.io

## How to

create new page

```
hugo new posts/new-post.md
```

Run server

```
hugo server
```

Deploy

```
git pull
```

if (any changes on /root)

```
git add . && git commit -m "update" && git push origin master
```

Update all changes via forestry

```
./deploy.sh
```
